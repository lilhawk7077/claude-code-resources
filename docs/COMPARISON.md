# Vexilo — Comparison with similar tools

| Tool | What it is | How Vexilo is different |
| --- | --- | --- |
| [Claude Code docs](https://code.claude.com/docs/en/overview) | Official reference | Vexilo is the *navigation layer*, not the reference |
| [gstack](https://github.com/garrytan/gstack) | Spec-driven dev workflow | Vexilo *indexes* gstack alongside ECC / Auto Agent |
| [Everything Claude Code](https://github.com/affaan-m/everything-claude-code) | 200+ agents + skills config | Vexilo shows you which ones to use for *your* task |
| [Anthropic Skills](https://github.com/anthropics/skills) | Official skill library | Vexilo cross-references with other toolkits |
| [awesome-claude-code](https://github.com/hesreallyhim/awesome-claude-code) | Curated list (47k stars) | Vexilo is in it; they list the meta-resources, Vexilo ships the index |
| [Claude Code Cheat Sheet (storyfox)](https://cc.storyfox.cz/) | One-page printable reference | Vexilo is live + searchable + feeds into Claude |
| [Claude Code Ultimate Guide](https://github.com/FlorianBruniaux/claude-code-ultimate-guide) | Long-form documentation site | Vexilo is the navigation index; the Ultimate Guide is the deep-dive |

## When to use what

- **First time installing Claude Code?** → Docs
- **Want to know what agents exist?** → Vexilo
- **Want to learn a specific feature in depth?** → Claude Code Ultimate Guide / docs
- **Want to install a community skill?** → awesome-claude-code / anthropic skills repo
- **Want to know which skill to invoke?** → Vexilo (5-step workflow + scenarios)
- **Want to feed everything into your Claude session?** → Vexilo ("Teach Claude this handbook" button)

## Anti-comparison

Vexilo is **not** trying to replace:

- The official docs (those are authoritative)
- gstack / ECC / Auto Agent (those are content; Vexilo is the index)
- Awesome lists (those are meta-resources; Vexilo is one of the entries)

Vexilo *depends* on all of the above continuing to ship good primitives. As long as they grow, Vexilo's index grows.
